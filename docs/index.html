遊び方
  1. 「スタート」を押すとTKGがシャッフルを開始するよ！
  2. 「ストップ」を押すとシャッフルが止まるよ！
  3. 「リトライ」を押すともう一度遊べるよ！
  4. 「イージーモード」を選択するとTKGのシャッフル数が120から８に減るよ！
  5. 「目押しモード」を選択するとTKGのシャッフルがランダムじゃなく巡回になるよ！
  6. 同じTKGを３枚揃えると X にポストできるよ！
